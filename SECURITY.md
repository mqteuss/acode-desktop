# Política de segurança

## Relatar uma vulnerabilidade

Não abra uma issue pública contendo tokens, URLs MCP privadas, caminhos locais, credenciais ou detalhes que permitam exploração.

Envie um contato privado para [@mqteus07](https://x.com/mqteus07) com:

- versão do ACode;
- versão do Windows;
- descrição objetiva do problema;
- passos mínimos para reproduzir;
- impacto observado;
- arquivos ou capturas sem dados sensíveis.

## Downloads oficiais

Considere oficiais somente os arquivos anexados às Releases de `mqteuss/acode-desktop` e os links apresentados pelo site publicado a partir deste repositório.

Sempre valide o SHA-256 usando o `SHA256SUMS.txt` da mesma release.
